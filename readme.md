# Splitwise Totals

## Prerequisites

- [NodeJS](https://nodejs.dev/)
- [Yarn](https://yarnpkg.com/)

## Installation

1. Register a new application [here](https://secure.splitwise.com/apps).

2. Copy `.env.example` to `.env`.

3. Fill in your API-key (step 1) and your User-ID in `.env`.


## Commands

```bash
yarn start [--fetch]
```
